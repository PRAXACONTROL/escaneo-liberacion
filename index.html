Configuración de lectura

El código de barras de la etiqueta RIVULIS trae el mismo Work Order (WO) repetido en los 16 rollos de una orden — no hay un identificador único por rollo. Cada escaneo válido cuenta como un rollo más. Este ajuste evita contar dos veces el mismo rollo si el operador lo escanea por error.
